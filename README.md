# Administrador de Campeonato F1

Sistema de gestión por consola para administrar los datos de telemetría y puntuación de una temporada de Fórmula 1, simulando el sistema utilizado por los equipos.

# Características Principales

* Interfaz de usuario interactiva por consola.
* Gestión (ABM) de Pilotos y Escuderías.
* Registro y validación de tiempos de carrera (formato `HH:MM:SS.mmm`).
* Cálculo automático de la Tabla de Posiciones mediante matrices bidimensionales.
